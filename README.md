# Бот с проверкой обязательной подписки

Простой Telegram-бот на Python. Бот удаляет сообщения пользователей, которые не подписались на обязательный канал/чат.

## Как это работает
1. Добавьте бота в нужный чат и выдайте права администратора (нужны права на удаление сообщений).
2. Укажите канал/чат, на который нужно обязательно подписаться.
3. Бот проверяет подписку и удаляет сообщения, если пользователь не подписан.

## Настройка

### 1. Установите зависимости
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### 2. Установите переменные окружения
- `BOT_TOKEN` — токен бота от @BotFather
- `REQUIRED_CHAT` — @username канала/чата или numeric ID (например, `@mychannel`)

Пример:
```bash
export BOT_TOKEN="123456:ABCDEF"
export REQUIRED_CHAT="@mychannel"
```

### 3. Запуск
```bash
python main.py
```

## Важно
- Бот должен быть админом в целевом чате (чтобы удалять сообщения).
- Для проверки подписки в канале лучше добавить бота админом в канал (иначе `getChatMember` может не видеть пользователей).
